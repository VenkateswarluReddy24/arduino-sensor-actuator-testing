# arduino-sensor-actuator-testing
# 🔧 Arduino Sensor & Actuator Testing Library

## 📌 Overview
This repository contains basic testing codes for commonly used sensors and actuators in robotics and IoT projects.

These modules are used for hardware validation before integrating into full systems.

---

## 📦 Modules Included

### 📏 Ultrasonic Sensor (HC-SR04)
- Distance measurement
- Used in obstacle avoiding robots

### ⚙️ Servo Motor
- Angle control testing
- Used in robotics arms and automation

### 📡 RFID Module (RC522)
- Card UID detection
- Used in authentication systems

### 🔌 Relay Module
- AC/DC load control
- Used in home automation

### 💡 LED
- Basic output testing

### 🔊 Buzzer
- Sound alert system

---

## 🛠️ Technologies Used
- Arduino UNO / NodeMCU / ESP32
- Embedded C (Arduino IDE)

---

## 🎯 Purpose
- Hardware testing
- Debugging components
- Learning embedded systems fundamentals
