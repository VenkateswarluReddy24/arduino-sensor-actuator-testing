1. Push 2

## 🚀 Features

- ✅ Arithmetic operations (+, -, ×, ÷)  
- ✅ Operator precedence (BODMAS)  
- ✅ Decimal number support  
- ✅ Scientific functions:
  - Square (hold C)
  - Cube (hold D)
  - Square root (hold A)  
- ✅ Input validation (invalid expressions blocked)  
- ✅ Error handling (divide by zero, invalid input)  
- ✅ Debounced keypad input  
- ✅ Hold-based advanced functions  

---

## ⚠️ Error Handling

The system detects and prevents:

- Invalid expressions (`+*5`, `5++2`)  
- Division by zero  
- Invalid numerical results (NaN / Infinity)  

Displays:
Math Error
