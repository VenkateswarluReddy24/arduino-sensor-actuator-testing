# 💧 Smart Water Tank Automation System (IoT + Safety Engine)

## 📌 Overview
This project implements an industrial-grade water tank automation system using ESP8266, ultrasonic sensing, and Blynk IoT.

The system ensures safe and efficient water management with multiple protection layers.

---

## 🧠 System Architecture

Ultrasonic Sensor → Control Logic → Safety Engine → Relay → Pump  
                               ↓  
                        Blynk + LCD + LEDs

---

## 🛠️ Hardware Components

| Component | Quantity |
|----------|---------|
| ESP8266 (NodeMCU) | 1 |
| Ultrasonic Sensor (HC-SR04) | 1 |
| Relay Module | 1 |
| LEDs | 4 |
| LCD (I2C) | 1 |

---

## ⚙️ Working Principle

- Distance indicates water level  
- Pump turns ON/OFF automatically  
- Safety checks prevent failures  
- Data displayed on LCD & Blynk  

---

## 🚀 Features

- 💧 Automatic water level control  
- ⚠ Overflow protection  
- 🔥 Dry-run protection  
- 🔄 Restart delay safety  
- 📱 IoT monitoring (Blynk)  
- 📟 LCD + LED indicators
