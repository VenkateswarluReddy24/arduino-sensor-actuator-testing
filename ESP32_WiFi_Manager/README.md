# 🌐 ESP32 WiFi Connection Manager

## 📌 Overview
This project implements a robust WiFi connection manager for ESP32, designed with industry-level reliability and fault tolerance.

---

## 🧠 System Architecture

ESP32 → WiFi Module → Router → Network Services

---

## ⚙️ Features

- 🔄 Auto-reconnect mechanism  
- ⏱ Timeout-based connection handling  
- 📡 Real-time IP monitoring  
- 🧠 Non-blocking design  
- 🔐 Secure credential handling  

---

## 🛠️ Hardware

| Component | Quantity |
|----------|---------|
| ESP32 | 1 |

---

## 💻 Software

- Arduino Framework (ESP32 Core)

---

## 🚀 Working Principle

- Attempts WiFi connection with timeout  
- If failed → retries automatically  
- Continuously monitors connection status  
- Prints diagnostics over Serial  

---

## 📊 Serial Output

- Connection progress  
- IP address  
- Reconnection logs  

---
