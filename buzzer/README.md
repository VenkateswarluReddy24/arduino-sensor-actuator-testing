# 🔊 Buzzer Test Module

## 📌 Description
Test for buzzer sound output using digital signals.

## 🔌 Pin Configuration
| Component | Pin |
|----------|-----|
| Buzzer   | D3  |

## ⚙️ Working
The buzzer turns ON and OFF at intervals to produce sound.

## 📤 Output
Beeping sound every 500ms.

## 🎯 Use Case
Used in alert systems and notifications.
