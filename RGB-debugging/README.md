# 🌈 RGB LED Control System (PWM Based)

## 📌 Overview
This project demonstrates precise control of an RGB LED using PWM signals from an Arduino.

It includes both static color control and smooth color transitions, showcasing embedded control techniques.

---

## 🛠️ Hardware Components

| Component | Quantity |
|----------|---------|
| Arduino Uno / Nano | 1 |
| RGB LED (Common Cathode) | 1 |
| Resistors (220Ω) | 3 |

---

## 🔌 Connections

| LED Pin | Arduino |
|--------|--------|
| Red | D9 |
| Green | D10 |
| Blue | D11 |
| Common | GND |

---

## ⚙️ Working Principle

- PWM signals control LED brightness  
- Different combinations produce colors  
- Smooth transitions are generated using incremental loops  

---

## 🧠 Embedded Logic

- `setColor(r,g,b)` controls RGB values  
- Fade algorithm creates smooth transitions  
- Delay controls transition speed  

---

## 🚀 Features

- 🎨 Full RGB color control
