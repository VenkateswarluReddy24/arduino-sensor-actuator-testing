# 🌍 Smart Web-Based World Clock (ESP8266 + NTP)

## 📌 Overview
This project implements a web-controlled global clock system using ESP8266.

Users can dynamically switch between cities via a web interface, with real-time updates displayed on an LCD.

---

## 🧠 System Architecture

ESP8266 → WiFi → NTP Server  
        → Web Server → User Control  
        → LCD Display

---

## ⚙️ Features

- 🌍 Multi-city time display  
- 🌐 Web-based control panel  
- ⏱ Real-time NTP synchronization  
- 📟 LCD display output  
- ⚡ Non-blocking system design  

---

## 🛠️ Hardware Components

| Component | Quantity |
|----------|---------|
| ESP8266 (NodeMCU) | 1 |
| 16x2 I2C LCD | 1 |

---

## 💻 Software

- Arduino IDE (ESP8266 Core)
- Embedded C

---

## 🚀 Working Principle

- ESP8266 connects to WiFi  
- Syncs time using NTP server  
- Hosts a web interface  
- User selects city  
- Time offset updates dynamically  
- LCD displays real-time clock
