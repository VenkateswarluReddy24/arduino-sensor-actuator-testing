# 📏 Ultrasonic Sensor (HC-SR04)

## 📌 Description
Measures distance using ultrasonic waves.

## 🔌 Pin Configuration
| Component | Pin |
|----------|-----|
| VCC      | 5V  |
| GND      | GND |
| TRIG     | D6  |
| ECHO     | D7 |

## ⚙️ Working
1. Sends ultrasonic pulse
2. Receives echo signal
3. Calculates distance

## 📤 Output
Distance displayed in Serial Monitor.

## 🎯 Use Case
Obstacle detection in robotics systems.
