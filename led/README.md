# 💡 LED Test Module

## 📌 Description
Basic LED blinking test to verify digital output functionality.

## 🔌 Pin Configuration
| Component | Pin |
|----------|-----|
| LED      | D2  |

## ⚙️ Working
The LED is turned ON and OFF every 1 second using digital output.

## 📤 Output
LED blinks continuously.

## 🎯 Use Case
Used to verify GPIO output before integrating into larger systems.
