# ✋ Gesture Controlled Media System (Ultrasonic + Python Automation)

## 📌 Overview
This project implements a real-time gesture recognition system using ultrasonic sensors.

Hand gestures are detected and translated into media control commands using Python automation.

---

## 🧠 System Architecture

Ultrasonic Sensors → Arduino Processing → Serial Communication → Python → System Control

---

## 🛠️ Hardware Components

| Component | Quantity |
|----------|---------|
| Arduino Uno | 1 |
| Ultrasonic Sensors (HC-SR04) | 2 |

---

## 💻 Software Stack

- Arduino (Embedded C)
- Python
- PyAutoGUI (Automation)

---

## ⚙️ Working Principle

- Two ultrasonic sensors detect hand position  
- Distance ranges mapped to gestures  
- Majority filter ensures stability  
- Python script converts gestures into system commands  

---

## ✋ Gesture Mapping

| Gesture | Action |
|--------|-------|
| Play | Play/Pause |
| Forward | Skip Forward |
| Rewind | Skip Back |
| Vup | Volume Up |
| Vdown | Volume Down |
| Stop | Stop |
